# Prediction Examples

This folder contains forecasting and regression examples for time series, using classical and machine-learning models.

- [ts_arima.md](ts_arima.md) — ARIMA modeling with automatic order selection and multi-step forecasting.
- [ts_conv1d.md](ts_conv1d.md) — 1D convolutional neural network for sequence-to-one regression.
- [ts_elm.md](ts_elm.md) — Extreme Learning Machine (randomized hidden layer) for fast regression.
- [ts_integtune.md](ts_integtune.md) — Integrated pipeline with hyperparameter tuning for model selection.
- [ts_knn.md](ts_knn.md) — k-Nearest Neighbors regression on sliding windows.
- [ts_lstm.md](ts_lstm.md) — LSTM recurrent neural network for sequence forecasting.
- [ts_mlp.md](ts_mlp.md) — Multilayer Perceptron regression on windowed features.
- [ts_rf.md](ts_rf.md) — Random Forest regression for nonlinear relationships.
- [ts_svm.md](ts_svm.md) — Support Vector Regression (SVR) for robust non-linear modeling.
- [ts_tune.md](ts_tune.md) — Hyperparameter tuning utilities for time series models.
